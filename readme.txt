# How to run?

- You need Node and NPM installed on your machine.
- Run `npm install` to install dependencies.
- Run `npm run dev` to run the project locally.
- Run `npm run build` to run build the project for deployment. A `dist` folder will be created.
- Run `npm run preview` to preview your build.

License : You can use this codebase how ever you'd like except for redistributing it as is or with very little modification (ex: just changing the name of variables, etc...).
